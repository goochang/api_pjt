### Django DRF Project
> 작업 완료일 : 2024/12/25
- 개요: Postman 테스트를 기반으로 제작된 django DRF 프로젝트입니다.

### API 명세서
- 회원가입  : /api/accounts 
- 로그인 : /api/accounts/login 
- 프로필 조회 : /api/accounts/<str:username> 
- 상품 등록 : /api/products
- 상품 목록 조회 :/api/products
- 상품 수정 : /api/products/<int:productId>
- 상품 삭제 : /api/products/<int:productId>
